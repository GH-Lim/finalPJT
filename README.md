## 실제 구현 정도

계획 및 구현 정도

- 제목별 검색 - 완성

- 장르별 검색 - 완성

- 선호 장르 별 추천 - 미완성
- 유저 정보 - 미완성
- 평점 작성 및 삭제, 수정 - 작성 완성 / 삭제, 수정 미완성
- 영화 좋아요 - 완성
- 유저 팔로우 - 미완성
